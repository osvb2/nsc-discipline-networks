How to read the maps

Each node represents a code. Node size usually indicates code frequency or weight. Links show code relations, co-occurrence, comparison bridges, or interpretive bridges, depending on the specific network. Hover over nodes and links to read definitions, participant details, evidence anchors, and bridge explanations.

Published through GitHub Pages from the main branch using the repository root.
